# ⚔️ VexVisualsPlus  
### *Визуальный PvP-мод для Minecraft*  

[![Forge](https://img.shields.io/badge/Forge-1.8.9—1.20.1-brightgreen)](https://files.minecraftforge.net/)
[![License](https://img.shields.io/github/license/yourname/VexVisualsPlus)](LICENSE)
[![Download](https://img.shields.io/badge/Download-latest-blue)](https://github.com/yourname/VexVisualsPlus/releases)

**VexVisualsPlus** — это мощный клиентский мод, который улучшает визуальную составляющую PvP-сражений.  
Анимации урона, цветные индикаторы, настраиваемый прицел и многое другое — всё для комфортной игры на серверах.

---

## ✨ Особенности

- 🎯 **Динамический прицел** — меняет цвет и форму при наведении на игрока / моба.  
- 💥 **Визуал урона** — реалистичные частицы крови и вспышки при попадании.  
- 📊 **Индикатор здоровья** — отображение HP противника над головой.  
- 🧪 **Эффекты зелий** — компактные иконки с таймерами действия.  
- 🔔 **Оповещения** — уведомления о низком здоровье, смене брони и комбо.  
- ⚙️ **Полная настройка** — конфигурация через файл `VexVisualsPlus.json` или внутриигровое меню (нажмите `V`).  
- 🚀 **Оптимизация** — минимальное влияние на FPS.

---

## 📥 Установка

1. Установите **Minecraft Forge** для нужной версии (1.8.9 / 1.12.2 / 1.16.5 / 1.20.1).  
2. Скачайте последнюю версию мода из [Releases](https://github.com/yourname/VexVisualsPlus/releases).  
3. Поместите `.jar` файл в папку `mods` (`.minecraft/mods`).  
4. Запустите игру с профилем Forge.  

> **Примечание:** Мод работает только на **клиенте** и не требует установки на сервере.

## 🎮 Использование

| Клавиша | Действие |
|---------|----------|
| `V`     | Открыть меню настройки мода |
| `B`     | Включить/выключить индикатор здоровья (если не настроено иное) |

Все параметры можно изменить в файле конфигурации:  
`.minecraft/config/VexVisualsPlus.json`
